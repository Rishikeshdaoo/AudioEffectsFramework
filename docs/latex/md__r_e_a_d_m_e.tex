Cross platform C++ library with low-\/latency audio effects processing algorithms designed for real-\/time audio processing.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{Motivation}\label{md__r_e_a_d_m_e_autotoc_md1}
This projects aims to provide an easy to use A\+PI for creating audio effects processing applications.
